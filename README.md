# Garment Order Tracking System

A professional, high-performance desktop application for tracking garment production orders, shipping schedules, and analytics. Designed for textile and garment manufacturing environments.

## Features
- **Dashboard Analytics**: Visual representation of shipping progress, trends, and country distribution.
- **Order Management**: Comprehensive tracking of order quantities, colours, and shipment statuses.
- **PDF/Excel Integration**: Import orders from PDF/Excel and export professional reports.
- **Automated Backups**: Built-in system for daily and on-close database backups.
- **Advanced Filtering**: Multi-criteria search for specific orders, colours, seasons, and date ranges.
- **Seamless Updates**: Integrated "Check for Updates" mechanism via GitHub.

## Tech Stack
- **Language**: Python 3.x
- **GUI**: Tkinter (with custom modern styling)
- **Database**: SQLite
- **Data Handling**: Pandas, Openpyxl, Pillow (PIL)

## Installation & Usage

1. **Clone the repository**:
   ```bash
   git clone https://github.com/rubeltechcom/garment-tracking-system.git
   ```

2. **Run Setup**:
   Execute `setup.bat` to install the required dependencies automatically.

3. **Launch the App**:
   Run `run.bat` to start the application.
